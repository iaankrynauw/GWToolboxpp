## Quick Start

### New to the codebase?
1. Read [CODEBASE_ARCHITECTURE.md](CODEBASE_ARCHITECTURE.md)
2. Review [MISSION_WIKI_IMPLEMENTATION.md](MISSION_WIKI_IMPLEMENTATION.md) as example
3. Keep [QUICK_REFERENCE.md](QUICK_REFERENCE.md) open while coding

### Adding a new feature?
1. Check [QUICK_REFERENCE.md](QUICK_REFERENCE.md) for templates
2. Follow patterns from [MISSION_WIKI_IMPLEMENTATION.md](MISSION_WIKI_IMPLEMENTATION.md)
3. Update [PROJECT_STATUS.md](PROJECT_STATUS.md) when complete

### Need a quick answer?
1. Check [QUICK_REFERENCE.md](QUICK_REFERENCE.md) first
2. Search [CODEBASE_ARCHITECTURE.md](CODEBASE_ARCHITECTURE.md) for details
3. Review [MISSION_WIKI_IMPLEMENTATION.md](MISSION_WIKI_IMPLEMENTATION.md) for examples

## Contributing

When adding new features:

1. **Document as you go** - Don't wait until the end
2. **Update relevant docs** - Keep information current
3. **Add examples** - Show, don't just tell
4. **Include rationale** - Explain why, not just what
5. **Test thoroughly** - Verify examples work
